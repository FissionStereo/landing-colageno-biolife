# Landing Colágeno Biolife

Landing page responsive para promocionar y vender Colágeno Hidrolizado PEPTANB&C - Collagen Peptides Biolife.

## Características

- Diseño responsive para desktop y móvil.
- Sección principal con producto destacado.
- Cards de beneficios y presentaciones.
- Botones de contacto por WhatsApp.
- Animaciones suaves e interacciones hover.
- Footer oscuro con información de contacto.

## Tecnologías

- HTML5
- CSS3
- JavaScript

## Vista previa

Página enfocada en una experiencia visual limpia, comercial y profesional para productos de bienestar y suplementos alimenticios.
